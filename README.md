## 🗂️ Projects

### 🐍 Session 1 — The First Hello
**Date:** May 1, 2026

The one that started everything. First time writing real Python.

**Concepts covered:**
- Variables — storing information
- `print()` — displaying output
- f-strings — combining text and variables cleanly

**First program ever written:**

    name = "Adrian"
    print(f"Hello, {name}! Welcome to Python.")

**Key rule learned:**
> `{}` is for SHOWING — inside f-strings only.

---

### 🐍 Session 2 — The Escalation Calculator
**Date:** May 2, 2026 (Bullwinkle shift 💪)

First real-world program. Connected directly to Apple support work.

**Concepts covered:**
- `input()` — receiving information from the user
- `int()` — converting text to numbers
- Basic math operations — addition, subtraction, multiplication
- Combining input + math into a working tool

**Program built:** An escalation calculator — taking real numbers from a support shift and calculating escalation rates.

**Key rules locked in:**
> `()` is for DOING — calling functions.
> `{}` is for SHOWING — inside f-strings only.
